# CollabDocs - Real-Time Collaborative Editor

A full-stack real-time collaborative document editor built for Internship Task 3.

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js + Express
- **Real-time:** Socket.IO  
- **Database:** MongoDB

## Features
- Real-time collaboration between multiple users
- Auto-save every 2 seconds
- User presence indicators
- MongoDB document persistence

## How to Run
### Backend
cd server
npm install
node index.js

### Frontend
cd client
npm install
npm start
